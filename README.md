# Introducción:
Es una analizador facial capaz de identificar o verificar a un sujeto a través de una imagen, vídeo o cualquier elemento audiovisual de su rostro. En este proyecto usaremos las librerías necesarias del procesamiento de imágenes en la cual crearemos un programa que reconozca el rostro de una imagen y devuelve los rostros más similares en comparación a nuestra query o consulta.
